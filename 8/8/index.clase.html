Tipos de selectores avanzados

  • Por agrupado => Seleccionamos utilizando las comas.
    h1, p, a { background-color : red; }

  • Selectores por espacio => nos permiten tener seleccionados elementos que no le importen su proximidad, es decir puedo tener lo siguiente div p { color: blue; }

    • [attr$=value]
